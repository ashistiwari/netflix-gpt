# Netflix GPT

A React app with Tailwind CSS styling scaffolded from Create React App.

## Project Overview

This project is a starter React application built with:

- React 19
- Create React App
- Tailwind CSS
- PostCSS

It includes a simple layout and style setup using Tailwind utilities.

## Getting Started

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

Open `http://localhost:3000` to view the app in your browser.

### Build for production

```bash
npm run build
```

This creates an optimized production bundle in the `build/` folder.

## Tailwind CSS

Tailwind is configured in `tailwind.config.js` and the app imports Tailwind directives in `src/index.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

The content paths include all JSX and JS files under `src/`.

## Available Scripts

- `npm start` - start the development server
- `npm run build` - create a production build
- `npm test` - run the test suite
- `npm run eject` - eject CRA configuration (irreversible)
